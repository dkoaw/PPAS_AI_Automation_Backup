# lgt_sidebar_ui

灯光合成工具总控侧边栏，作为所有 LGT 原子技能的统一入口 UI。

## 功能描述
作为灯光环节的核心交互组件，该技能在 Blender 的 N-Panel（侧边栏）中集成了一个统一的控制台界面。它将所有的 LGT 原子技能串联起来，为灯光师提供一站式工作流。

## 面板布局
- **工程信息区**：显示项目名、镜头号等元数据。
- **渲染设置区**：集成了渲染引擎切换、参数一键设置等功能。
- **贴图审计区**：提供 VRAM 审计、分辨率优化功能的入口。
- **资源管理区**：快速打开外部贴图管理器、设置输出路径。

## 设计原则
- **操作原子化**：支持每个独立工具的单次调用。
- **流程向导化**：按照标准工作流顺序排列 UI 按钮。
- **状态回显**：在 UI 上直观反映当前场景的关键状态。
