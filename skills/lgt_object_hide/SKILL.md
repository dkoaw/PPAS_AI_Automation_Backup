# 🛠️ lgt_object_hide (物体分层隐藏技能) V6 Robust

## 技能描述
用于在灯光合成环节，针对特定渲染层（View Layer）精准控制物体的显示与隐藏。

## 核心特性 (V6 Robust)
1. **动画迁移保护**：自动识别并迁移物体已有的隐藏动画关键帧至 `original_hide_...` 属性，确保驱动器逻辑与原始动画完美融合（取最大值叠加）。
2. **列表存储机制**：使用 `hidden_layer_list` 字符串属性存储黑名单层名，支持通过 UI 属性面板直接观察和编辑屏蔽状态。
3. **5.0.1 深度适配**：支持 Blender 5.0 的 `ChannelBags` 动画架构，确保管线升级后的稳定性。
4. **智能开关逻辑**：支持一键 Toggle。若当前层已在黑名单，再次点击即可恢复显示。

## 导出 Operator
- `ppas.hide_selected_on_layer`: 针对当前层执行隐藏/显示切换（含动画迁移）。
