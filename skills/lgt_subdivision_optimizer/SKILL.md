# 🛠️ lgt_subdivision_optimizer (细分精度自动优化技能)

## 技能描述
基于物体与相机的视觉距离，自动优化全场景物体的表面细分（Subdivision Surface）级别。

## 优化逻辑 (分级规则)
系统采用焦距补偿算法（以 50mm 为基准）计算有效距离：
1. **近景 (有效距离 < 3m)**: 渲染细分 = 2, 视图细分 = 1。
2. **中景 (3m <= 有效距离 < 10m)**: 渲染细分 = 1, 视图细分 = 0。
3. **远景 (有效距离 >= 10m)**: 渲染细分 = 0, 视图细分 = 0。

## 核心特性
- **算法复用**：与贴图精度优化共享一套距离估算逻辑，确保管线标准统一。
- **性能平衡**：在保证渲染质量的前提下，最大限度降低视口预览压力。

## 导出 Operator
- `ppas.auto_subdiv_res`: 执行基于距离的细分自动优化。
