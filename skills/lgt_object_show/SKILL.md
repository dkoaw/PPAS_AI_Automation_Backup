# 🛠️ lgt_object_show (物体分层强制显示技能)

## 技能描述
用于在灯光环节，强制将当前活跃物体从渲染层的“隐藏黑名单”中移除，并立即刷新其显示状态。

## 核心特性
1. **名单精准剔除**：自动解析物体上的 `hidden_layer_list` 属性，并将当前活跃层名移除。
2. **强制更新序列 (Force Update)**：
   - **驱动器重激**：通过静默开关 `mute` 状态，强制 Blender 重新评估可见性驱动器。
   - **数据标记**：执行 `update_tag()` 确保对象数据状态同步。
   - **全局重绘**：触发窗口重绘，解决驱动器修改后视图不刷新的“延迟”问题。

## 导出 Operator
- `ppas.show_active_on_layer`: 强制在当前层显示活跃物体。
